# Duan Wenrui's dotfiles

## Requirements

+ [Vim 9.0](https://www.vim.org/vim90.php)

## Installation

```shell
# bash
cp bashrc ~/.bashrc
cp bash_aliases ~/.bash_aliases
cp bash_logout ~/.bash_logout

# git
cp gitconfig ~/.gitconfig

# ssh
cp sshconfig ~/.ssh/config

# vim
cp vimrc ~/.vimrc
```
